Accessibility Statement

We are committed to making this website accessible to all users, regardless of ability or the technologies they use.

This website has been designed and developed to comply with the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA. Accessibility has been considered throughout the design, content, and development process to ensure a clear, usable, and inclusive experience.

Compliance Status

This website is fully compliant with WCAG 2.1 AA standards.

Accessibility features

The website includes accessibility features such as:

  • Clear page structure and semantic headings.
  • Keyboard-accessible navigation and interactive elements.
  • Visible focus indicators.
  • Sufficient colour contrast
  • alternative text for meaningful images.
  • Responsive layouts that support different screen sizes and zoom levels.

Testing

The website has been tested using keyboard-only navigation and screen readers, as part of an accessibility review. It has also been checked for colour contrast, reflow at increased zoom levels, and usability across different devices and screen sizes.

Non-accessible content

At the time of this statement, we are not aware of any accessibility barriers on the website. If any issue is reported, we will investigate and address it promptly.

Alternative access

If you experience any difficulty using this website or need content in another format, please contact us by email or phone and we will provide support.
